Checkradio

Make classic checkbox and radio sexier with different colors, sizes, and states

View on Github
NPM Package 1.1.1
Colors Yes
Sizes Yes
Javascript No
Variables Yes

Use is-rtl to display a right-to-left design of checkbox or radio (text at left)


Don't forget to set the id attribute on input and the for attribute on label with the same value to get it work. This will link the label to the checkbox and then handle the click.


Please note that due to CSS limitations, the structure of the checkbox is different than the one provided as an example in Bulma's documentation.
(Input is not wrapped by the label)

Styles

Circle

You can add the is-circle modifier to the input to get a rounded checkbox


Block

You can add the is-block modifier to the input to get a full colored checkbox


No border

Adding the has-no-border modifier to the input will remove the checkbox border


Background colored

You can add the has-background-color modifier to the input to get the checkbox background color impacted
