Javascript

The extension comes with a JavaScript implementation to expand / collapse accordion items. Don't forget to include it in your project.

{% highlight javascript %}{{ accordion_js }}{% endhighlight %}

Then just add the following code once your DOM's ready:

{% highlight javascript %}var accordions = bulmaAccordion.attach(); // accordions now contains an array of all Accordion instances{% endhighlight %}
